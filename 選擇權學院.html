OPTIONS 101

選擇權是什麼CALL / PUT

選擇權是一份「權利」合約：買方付出權利金，取得在未來某個時點以特定價格買進或賣出股票的權利；賣方收取權利金，但承擔被履約的義務。

CCall（買權）

持有人有權「用約定價格買進」股票。當股價上漲，Call 的價值上升 — 因為你能用較便宜的價格買到更貴的股票。

舉例：台積電股價 $900，你持有履約價 $920 的 Call。若股價漲到 $960，你可以用 $920 買進，等於賺到 $40 的內含價值。

PPut（賣權）

持有人有權「用約定價格賣出」股票。當股價下跌，Put 的價值上升 — 因為你能用較高的價格賣出較便宜的股票。

舉例：股價 $900，你持有履約價 $880 的 Put。若股價跌到 $840，你可以用 $880 賣出，等於賺到 $40 的內含價值。

希臘字母 01 / 05

DeltaΔ · 股價敏感度

衡量「股價每變動 $1，選擇權價格大約會變動多少」，也約略等於該選擇權「到期時落在價內的機率」。

∂定義

Delta = ∂(選擇權價格) / ∂(股價)

Call 的 Delta 介於 0 ~ 1；Put 的 Delta 介於 -1 ~ 0。深度價外 ≈ 0，價平 ≈ 0.5（Call）或 -0.5（Put），深度價內趨近 ±1。

部位Delta 範圍股價漲時
買進 Call0 ~ +1獲利
賣出 Call-1 ~ 0虧損
買進 Put-1 ~ 0虧損
賣出 Put0 ~ +1獲利

例實例說明

情境
股票 XYZ 現價 $100，持有 Delta = 0.5 的 Call。

若股價上漲 $2 → 選擇權價格約上漲 $2 × 0.5 = $1.00。

若股價下跌 $2 → 選擇權價格約下跌 $1.00。

Delta 也是「避險比率」：賣方若要做 Delta 中性避險，每賣 1 口 Delta 0.5 的 Call（100股/口），須買進 50 股股票來對沖股價變動的風險，這就是造市商常用的 Delta Hedging。

經濟意涵

Delta 本質上回答了「這個部位有多像一張股票」：Delta = 0.5 的 Call，漲跌幅度大約是持有 50 股股票的一半，所以常被稱為「等效股數（share equivalent）」。對投機者，Delta 越高代表方向性曝險越大、越接近直接持有股票；對避險者與造市商，Delta 是計算「需要買賣多少股票才能讓整體部位不受股價漲跌影響」的關鍵指標，是選擇權市場流動性供給與風險管理的核心。

希臘字母 02 / 05

GammaΓ · Delta 的加速度

衡量「股價每變動 $1，Delta 會變動多少」，也就是「Delta 的變動率」，常被稱為選擇權的「加速度」。

∂定義

Gamma = ∂(Delta) / ∂(股價)

買方（不論 Call 或 Put）的 Gamma 永遠是正值；賣方永遠是負值。Gamma 在「價平（At-the-money）」最大，深度價內或價外則趨近於 0。

例實例說明

情境
股票 $100，買進價平 Call，Delta = 0.50，Gamma = 0.06。

股價漲到 $101 → 新 Delta ≈ 0.50 + 0.06 = 0.56（部位變得更看漲，漲速加快）。

股價跌到 $99 → 新 Delta ≈ 0.50 - 0.06 = 0.44（部位看漲程度減弱）。

經濟意涵

Gamma 描述的是「你的方向性曝險會如何隨股價自我增強」。買方是「正 Gamma」：股價往有利方向移動時，部位會自動變得更有利（Delta 加速擴大），股價往不利方向移動時，曝險自動縮小，這正是「買選擇權風險有限、報酬不對稱放大」的數學根源 —— 你不需要預測對「精確時機」，只要方向大致正確且波動夠大，正 Gamma 就會持續放大你的獲利。賣方是「負 Gamma」：部位會在不利方向上加速惡化，這也是為什麼選擇權賣方（尤其是裸賣）必須非常嚴格地控制部位大小與停損，因為風險會隨著價格走勢「加速」擴大，而非線性增加。Gamma 也是造市商避險成本的核心：高 Gamma 部位需要更頻繁地調整避險股數。

希臘字母 03 / 05

ThetaΘ · 時間價值流逝

衡量「每經過一天，選擇權價格會減少多少」，反映選擇權「會隨時間貶值」的特性。

∂定義

Theta = ∂(選擇權價格) / ∂(時間)

買方（Long Call / Long Put）的 Theta 通常為負值（每天都在貶值）；賣方的 Theta 為正值（每天都在獲利）。Theta 隨著接近到期日而加速，價平選擇權的 Theta 衰減最快（尤其最後 30 天）。

例實例說明

情境
買進 Call，現價 $5.00，Theta = -0.05。

若股價與波動度完全不變，明天這張 Call 理論上會剩下 $5.00 - $0.05 = $4.95。

對賣方而言，他每天能穩定收取約 $0.05 的時間價值，這就是賣方策略「躺著賺」的來源 —— 只要股價不大幅變動。

經濟意涵

Theta 反映選擇權的「租金本質」：買方付出權利金，等於是「租用」在到期前獲利的可能性，時間每過一天，這個機會視窗就少一天，價值自然遞減，這是選擇權有別於股票最關鍵的特徵 —— 股票放著不賣不會貶值，但選擇權會。賣方則扮演「出租者」的角色，只要時間流逝且股價未大幅偏離，就能持續賺取租金（時間價值）。這也解釋了為什麼「買方需要股價在有限時間內朝對的方向移動足夠幅度」才能獲利，光是「猜對方向」是不夠的，還必須「猜對時機」；而賣方的優勢正是時間站在他這一邊，這也是 Theta 與 Gamma 之間存在的經典抵換關係：正 Gamma（買方）必然伴隨負 Theta，負 Gamma（賣方）必然伴隨正 Theta。

希臘字母 04 / 05

Vegaν · 波動度敏感度

衡量「隱含波動度（IV）每變動 1%，選擇權價格會變動多少」，反映市場對未來不確定性的定價。

∂定義

Vega = ∂(選擇權價格) / ∂(隱含波動度)

買方（Call 或 Put）的 Vega 永遠是正值：波動度上升，選擇權變貴。賣方的 Vega 為負值：波動度上升會侵蝕賣方獲利。Vega 在價平選擇權、且距到期時間較長時最大。

例實例說明

情境
買進 Call，現價 $5.00，Vega = 0.12。財報公布前市場恐慌，IV 從 25% 跳升至 35%。

選擇權價格約上漲 0.12 × 10 = $1.20，即使股價完全沒動！

財報公布後恐慌解除（IV Crush），IV 從 35% 驟降回 22% → 即使股價照你預測的方向走，選擇權價格也可能因 Vega 損失而幾乎不漲，這是許多新手在財報前買選擇權卻虧錢的主因。

經濟意涵

Vega 衡量的是「市場對不確定性的定價」：隱含波動度越高，代表市場預期股價未來的波動幅度越大，選擇權（不論 Call 或 Put）的價值都會提高，因為「劇烈波動」對買方有利、對賣方不利。買方本質上是在「做多波動度」，即使方向看錯，只要市場恐慌情緒升溫（IV 飆升），也可能獲利；賣方則是在「做空波動度」，靠的是市場恐慌情緒最終會回歸平靜（Mean Reversion）。這正是為什麼選擇權定價不只是賭「漲或跌」，更是賭「漲跌的劇烈程度」，Vega 是連結選擇權市場與總體不確定性（如地緣政治、利率決策、財報）最直接的橋樑。

希臘字母 05 / 05

Rhoρ · 利率敏感度

衡量「無風險利率每變動 1%，選擇權價格會變動多少」，影響通常最小，但在利率劇烈變動或長天期選擇權上不可忽視。

∂定義

Rho = ∂(選擇權價格) / ∂(無風險利率)

Call 的 Rho 為正值（利率升，Call 變貴）；Put 的 Rho 為負值（利率升，Put 變便宜）。對短天期選擇權影響極小，但對 LEAPS（長天期選擇權，一年以上）影響顯著。

例實例說明

情境
買進一年期 Call，Rho = 0.45。央行升息 1 碼（0.25%）。

選擇權價格約上漲 0.45 × 0.25 = $0.11。

直覺解釋：買 Call 相當於用很少的權利金「延遲付款」鎖定未來買進股票的價格，等於把原本要付的本金省下來放在無風險利率上生息；利率越高，這個「延遲付款的好處」越值錢，Call 因此越貴；Put 邏輯相反，因為持有 Put 等於延遲收到賣股票的錢，利率越高，等待的機會成本越大，Put 因此越便宜。

經濟意涵

Rho 提醒我們，選擇權定價不只是「股票替代品」，它背後也隱含一筆「資金的時間價值」交易：買 Call 等於延遲支付股票的全額成本，這筆被遞延的資金理論上可以拿去賺取無風險利率，因此利率越高，Call 的吸引力（與理論價格）越高；反之賣方收到賣股的款項被往後遞延，利率越高，等待的機會成本越大，故 Put 越便宜。在升息循環中，這也是為何法人會更偏好用 Call 取代直接持有股票（資金效率更高），而 Rho 正是量化這種「資金佔用成本」差異的工具。

進階 01 / 02 · GAMMA SCALPING

用期貨 / 個股做 Long GammaDELTA-HEDGED LONG GAMMA

期貨與股票本身沒有 Gamma（它們的損益是直線，沒有凸性），所以「純做多 Gamma」一定要從選擇權取得，期貨/股票的角色是動態沖銷 Delta，把方向性風險歸零，只留下對「波動本身」的曝險。

M機制：為什麼期貨/股票能「純化」出 Gamma

選擇權的 Delta 會隨股價移動而改變（這就是 Gamma），若你買進選擇權後完全不避險，部位同時帶有方向性風險（Delta）與波動性風險（Gamma），兩者混在一起。做法是：用期貨或股票放空（或買進）等量的 Delta，讓淨 Delta 維持在 0 附近，剩下的損益來源就只剩下「股價怎麼動」這件事本身 — 不管漲或跌，只要動得夠大，你就賺錢；不動，你就虧 Theta。這個操作俗稱「Gamma Scalping（伽瑪刮頭皮）」，是造市商與波動度交易者最核心的避險與獲利手法。

用期貨而非股票避險的好處：保證金效率高（不需付出股票全額本金）、放空不受借券限制、無股息調整問題；股票避險的好處則是流動性深、無到期日轉倉成本、和選擇權的標的完全一致（無基差風險）。指數選擇權常用期貨避險，個股選擇權則兩者皆可。

L建立部位

買進 1 口 Call 履約價 $100　權利金 $5　Delta 0.50　Gamma 0.06（每口=100股）
放空股票/期貨避險 放空 50 股（= Delta 50 股當量）讓淨 Delta ≈ 0

之後每天股價一變動，Delta 會跟著 Gamma 改變，你就要重新調整避險部位的股數/口數 — 這個「追著 Delta 調整避險」的動作，就是把 Gamma 轉換成實現損益的過程。

進階 02 / 02 · CALENDAR + DELTA HEDGE

用期貨 / 個股做 Long VegaDELTA-HEDGED CALENDAR SPREAD

期貨與股票同樣沒有 Vega（它們的價格不受「隱含波動度」影響），所以期貨/股票永遠無法直接做多 Vega，角色一樣只是拿來沖銷 Delta；真正的 Vega 曝險必須來自選擇權本身，而「縱向價差（Calendar Spread）」是業界用來把 Vega 從 Gamma 中相對分離出來的標準做法。

M機制：為什麼用 Calendar Spread 而不是單純買一個 Call

單純買一口 Call 並用股票/期貨避險 Delta（也就是上一頁的做法），會同時得到正 Gamma + 正 Vega，兩者綁在一起，無法只賺波動度上升的錢而不被股價亂動干擾。若改用「買進長天期 ATM 選擇權 + 賣出短天期同履約價 ATM 選擇權」（Calendar Spread）：長天期選擇權的 Vega 較大、Gamma 較小；短天期選擇權的 Vega 較小、Gamma 較大。兩者相減後，淨 Gamma 會被壓得很小（甚至略為負值），但淨 Vega 仍維持顯著正值 — 這樣建構出來的部位，對「股價亂動」相對不敏感，卻仍然高度暴露在「隱含波動度上升」之下，是更接近「純粹做多波動度本身（IV）」的結構。建好部位後，一樣用股票或期貨把殘餘的淨 Delta 沖銷掉。

把這頁跟前兩頁對照：單純 Long Gamma（避險 Call）那一頁同樣的價格路徑只賺了 $170；這裡用跨式同時持有 Gamma 與 Vega，光 Gamma 部分就因為部位放大一倍而賺到 $390，再加上 IV 同步上升額外貢獻的 $120 Vega 損益，扣掉雙倍的 Theta 成本 $50 後，淨賺 $460 — 遠高於只賺單一種波動度。這就是「Long Vol」的精髓：不挑波動的來源，股價亂動也好、市場恐慌也好，兩種波動度收入一起收。

經濟意涵

「Long Vol」交易者本質上是在向市場銷售一份「萬用不確定性保險」的相反方— 他們不是出售保險的人，而是用權利金「買進」對不確定性的曝險，並用期貨/個股把唯一不想要的部分（方向）濾掉，只留下「世界變得更不確定」這件事本身的純粹收益來源。經濟上，這對應到「波動度風險溢酬（Volatility Risk Premium）」的另一面：市場上多數時候，賣方（出售保險者）因為承擔尾端風險而長期享有正期望值，但在實現波動真正超出市場原先定價的少數時刻（如金融危機、政策意外、地緣衝突），買方（Long Vol）會獲得巨大、不對稱的回報。專業波動度交易台正是靠著用期貨/個股把方向性風險完全剝離，讓部位變成一個「對總體不確定性程度」最純粹的曝險工具，再透過嚴格控制 Theta 持有成本與部位規模，等待市場真正劇烈變動的時刻獲取超額報酬，這也是為什麼長期持有的 Long Vol 部位常被視為一種「尾端風險避險（Tail Hedge）」，用來對沖投資組合中其他部位在極端行情下的虧損。
